# scalr-ctl is a command-line interface to your Scalr account

Version 1.0 works with the new Scalr JSON API.

Install:

```console
pip install scalr-ctl
```

Configure:

```console
scalr-ctl configure
```

Use:

```console
scalr-ctl --help
scalr-ctl <tab> <tab>
```

Update:

```console
pip install --upgrade scalr-ctl
scalr-ctl update
```
